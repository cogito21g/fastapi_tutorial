## Setting

- 가상환경설정
```bash
conda create -n fastapi python=3.10
conda activate fastapi
```

- 패키지 설치
```bash
pip install fastapi
pip install uvicorn
pip install jinja2
pip install python-multipart
pip install sqlalchemy
pip install email-validator
```

- 실행
```bash
python app.py
```
